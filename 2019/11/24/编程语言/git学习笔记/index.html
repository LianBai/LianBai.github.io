    1. git常用命令

    git常用命令

    1. 初始化仓库，可以将本地目录变成git可以管理的本地仓库 

       git init

    2. 关联远程仓库，可以将本地仓库与远程仓库关联，切记是本地仓库，不是本地目录，可以先利用git init将本地目录编程本地仓库后再进行关联。 

       git remote add origin 仓库地址

    3. 取消关联地址

       git remote rm origin

    4. 将远程仓库与本地仓库同步合并，下列命令是合并master分支

       git pull --rebase origin master

    5. 查看远程分支命令和查看本地分支

       git branch -a    //查看远程分支
 git branch        //查看本地分支

    6. 切换分支命令

       git checkout 分支名字

    7. 查看仓库状态

       git status

    8. 将工作区内容添加到缓存区

       git add .        //所有内容添加
 git add            //选择内容添加

    9. 缓存区内容提交到本地仓库区

       git commit -m "提交注释"

    10. 把本地仓库代码内容推送到远程仓库 

      git push -u origin master

    11. 查看日志(历史版本)

      git log

git reflog

    12. 撤销修改
      — 撤销文件在工作区的修改: 

      git checkout -- 文件名
git checkout

      — 撤销文件在缓存区的修改: 

      git reset 文件名          # 撤销指定文件的git add操作，即在缓存区的修改
git reset                # 撤销之前所有的git add操作，即在缓存区的修改
